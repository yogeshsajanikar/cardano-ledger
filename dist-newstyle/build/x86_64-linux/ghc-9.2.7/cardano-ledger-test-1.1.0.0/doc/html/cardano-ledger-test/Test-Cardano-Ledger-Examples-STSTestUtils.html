class AlonzoBased era failure where Source #

We use this to write set of tests that raise AlonzoBased PredicateFailures. The type we use is for '(fail ~ PredicateFailure (EraRule UTXOW era))' . There are 4 types of AlonzoBased PredicateFailures: UtxowPredFail, UtxosPredicateFailure, UtxoPredicateFailure, and UtxowPredicateFailure . The idea is to make tests that only raise these failures, in Alonzo and future Eras.
